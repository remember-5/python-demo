## 介绍
项目是平常使用和一些练习的demo


| 模块 | 介绍 | 
| -- | -- |
| email | py发送邮件|
| music | 音乐爬虫|
| mysql | mysql的连接和工具 |
| thread | py多线程 |
| video | 视频爬虫 |
| utils | 工具包 |
| face | 人脸识别 |


## 安装
建议使用conda+uv
```bash
conda create --name python-demo python==3.11
conda activate python-demo
pip install uv
uv pip install .
```
